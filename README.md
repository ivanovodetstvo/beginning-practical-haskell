# Beginning Practical Haskell

An introductory Haskell programming course by [Richard Cook][rcookdotorg]

## Course contents

[Start here](index.md)

## Generating the course notes

You'll need [Stack][stack] and [Pandoc][pandoc]:

```bash
stack install pandoc
```

### Linux/OS X

```bash
make
```

### Windows

```cmd
.\make.cmd
```
